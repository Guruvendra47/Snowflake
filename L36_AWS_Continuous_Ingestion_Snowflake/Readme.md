This project contains hands-on materials, SQL scripts, AWS policies, and documentation for implementing **Snowflake continuous and incremental data ingestion using AWS S3 external stages, Streams, Tasks, and File Formats**.

The purpose of this repository is to help learners understand real Snowflake data engineering workflows using cloud storage, secure access, and CDC automation.
